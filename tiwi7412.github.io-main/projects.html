Adversarial Data Attack Detection for L3Harris Machine Learning Model

Background - L3Harris Technologies works alongside the US Space Force to track information about the orbits of satellites, rockets, and other space debris. In order to reduce the work orbital analysts have to do, L3Harris Technologies wants to develop a robust ML model that is resistant to different kinds of attacks by potential adversaries, that is tolerant to mistakes made when classifying incoming data, and that can explain decisions to orbital analysts that haven't worked with the model.

Problem Statement - Our neural network model will predict historical stock market trends for DJIA, S&P 500, and NASDAQ. Using this existing model we will adapt it to fit the needs of continuous training, anomaly detection, protection against adversarial attacks, as well as explainability of model predictions in the use case of orbital data.

Results - We have achieved outlier detection on data using the methods: Local Outlier Factor, KNN, Hurst Exponents, Hole detection. We simulated continuous retraining on data with the use of window parsing. Additionally, we simulated attacks by using random data generation. Our project includes a fully implemented pipeline outputting all significant results to StreamLit UI dashboard.

Battleship in QTCreator

I created this game for a project in my Intensive programming workshop class. I used QTCreator to manage the UI. This was coded in C++ and taught me how to handle signals in C++. Two players can play and it hides the screen of the opposing player while they are active. The brown shapes are ships, the gold are treasure chests that allow the player to shoot twice, and the black ones are mines that if hit will hit the shooting players board.

Shopping robot controller

For my intro to robotics course, our final project involved creating a controller for a robot. The robot autonomously mapped a grocery store and then navigated the store to find objects. It used color blob detection to locate golden blocks. It then used inverse kinematics to control the arm in order to grab the blocks and drop them in a basket. This taught me a lot and challenged me as well. I needed to use a lot of math in order to move the arm and draw a map of the store. It also taught me about color blob detection and how to code that.
